Installation and Setting Up of Python 
Steps to Install Python 
To begin programming in Python: 
● Go to the official website: python.org 
● Download the latest version for your operating system. 
● Launch the installer and ensure you check the "Add Python to PATH" option. 
● Open your terminal or command prompt and type:
python --version

Python Virtual Environment
A virtual environment is an isolated Python workspace that helps manage dependencies for different projects. It ensures that packages for one project do not interfere with another.
creating vitual env - python -m venv env_name
activating virtual env - env_name/Scripts/activate
installing new packages - pip install package_name
deactivating - deactivate
